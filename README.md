# Blue Lagoon Resort

A full MERN stack hotel website for Blue Lagoon Resort.

## Structure

- `client/` - React + Vite frontend
- `server/` - Express + MongoDB backend

## Run locally

1. Install dependencies in `client/` and `server/`.
2. Copy `server/.env.example` to `server/.env` and set your values.
3. Run the backend with `npm run dev` inside `server/`.
4. Run the frontend with `npm run dev` inside `client/`.
